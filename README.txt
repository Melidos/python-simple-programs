# python-simple-programs
Simple and easy programs in Python to learn.

convertCSVToKeychainFormat.py : Convert a Dashlane csv exported file to an Apple keychain format.
divisors.py : Returns a list of all the divisors of a number.
getYear100.py : Return the year in which you will be 100 years old.
oddOrEven.py : Returns if a number is odd or even.
